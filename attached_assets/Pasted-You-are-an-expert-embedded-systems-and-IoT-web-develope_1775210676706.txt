You are an expert embedded systems and IoT web developer.

Generate a COMPLETE, production-quality ESP8266 web dashboard for a Smart Water Management System. The system communicates with an Arduino via Serial and must be fully functional, efficient, and mobile-friendly.

========================
SYSTEM REQUIREMENTS
===================

Hardware:

* ESP8266 (NodeMCU)
* Arduino (sending serial data)
* Data format received over Serial:
  "LEVEL:75;DIST:25;DRY:0;OVERFLOW:1;"

Where:

* LEVEL = water level percentage (0–100)
* DIST = ultrasonic distance (cm)
* DRY = 1 if dry run detected, else 0
* OVERFLOW = 1 if overflow detected, else 0

ESP8266 must:

* Host a web server
* Serve a modern dashboard UI
* Handle pump ON/OFF commands via HTTP routes
* Communicate commands to Arduino via Serial ("APP_ON", "APP_OFF")

========================

   * Animated progress bar (0–100%)

4. ANIMATED WATER TANK VISUAL

   * Use CSS animation (no heavy libraries)

5. CONTROL PANEL

   * Two large buttons:

     * Turn Pump ON (green)

     /on and /off routes

6. STATUS INDICATORS

   * Dry Run:

     * Red if active

     * Red if active

7. LIVE DATA DISPLAY

   * Water level (%)
   * Distance (cm)

8. AUTO-REFRESH (REAL-TIME)

   * Use JavaScript fetch() to call /data endpoint every 2 seconds
   * Update UI dynamically without reloading page

========================
BACKEND (ESP8266 CODE)
======================

Use:

* ESP8266WebServer library

Implement:

* "/" → serves dashboard HTML
* "/data" → returns JSON:
  {
  "level": 75,
  "distance": 25,
  "dry": 0,
  "overflow": 1
  }
* "/on" → sends "APP_ON" to Arduino
* "/off" → sends "APP_OFF" to Arduino

Include:

* Function to parse Serial input safely
* Avoid memory fragmentation (use String carefully)
